# Server Performance Stats

Bash script to analyze basic server performance stats — CPU usage, memory usage, disk usage, and top processes by CPU/memory.

## Project URL
https://roadmap.sh/projects/server-stats

## How to run
```bash
chmod +x server-stats.sh
./server-stats.sh
```

## What it covers
- Total CPU usage
- Total memory usage (used/free + percentage)
- Total disk usage (used/free + percentage)
- Top 5 processes by CPU usage
- Top 5 processes by memory usage
- Stretch: OS version, uptime, load average, logged in users, failed login attempts
